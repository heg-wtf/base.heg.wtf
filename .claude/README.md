# heg-wtf 조직 개발 가이드

heg-wtf 조직의 공식 개발 규칙과 표준을 정의하는 저장소입니다.

## 개요

이 저장소는 heg-wtf 조직에서 진행하는 모든 프로젝트에 적용되는 개발 가이드를 관리합니다.

## 문서

### [claude.md](./claude.md)

조직의 개발 표준을 정의한 메인 문서입니다. 다음 내용을 포함합니다:

- **기술 스택**: Supabase, Flutter, GitHub Pages 기반 개발 환경
- **Git Commit 규칙**: Gitmoji 방식의 커밋 메시지 표준
- **브랜치 전략**: main 브랜치 기반 워크플로우
- **코드 스타일**: 약어 지양, full text 사용 원칙
- **테스트 정책**: 모든 코드에 대한 테스트 필수 작성
- **문서화 가이드**: README, API 문서 작성 기준

## 사용 방법

모든 팀 멤버는 새 프로젝트 시작 전 또는 기여 전에 `claude.md` 문서를 숙지해야 합니다.
